PPS V1 — vairāki darbinieki pie viena paneļa

Iekļauts:
- katrs darbinieks savā telefonā izvēlas savu vārdu vienu reizi;
- vienam darbiniekam vienlaikus tikai viena aktīva darba sesija;
- pie viena paneļa var strādāt vairāki darbinieki vienlaikus vai secīgi;
- katram darbiniekam atsevišķs sākums, pauze, turpināšana un beigu laiks;
- darbinieks pabeidz tikai savu darbu;
- paneli pabeidz administrācija, kad nav aktīvu darbinieku;
- administratora piešķīrumi vairākiem darbiniekiem;
- nepiešķirtu paneli var izvēlēties jebkurš tās rūpnīcas darbinieks;
- dzīvais skats;
- pārskats un CSV eksports;
- objektu un paneļu Excel imports.

GitHub aizvieto:
- index.html
- styles.css
- app.js

Svarīgi:
V1 izmanto jaunu Firestore kolekciju sessions.
Iepriekšējā jobs kolekcija netiek dzēsta, bet V1 to neizmanto.
